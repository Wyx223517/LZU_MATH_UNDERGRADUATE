# 📚 拓扑学 (Topology)

> 2022级 | 大三下学期 | 必修课 | 4学分

这是一份关于兰州大学2022级 **Topology** 课程的非官方学习回顾。

## 👨‍🏫 课程信息

- **授课教师:** 陈立志 讲师
- **学期:** 大三下学期
- **课程性质:** 专业必修课
- **学分:** 4

## 📝 成绩构成

本课程的最终成绩由以下两部分构成，各占 $50\%$：

- **平时分 (50%)**: 包括平时作业和期中考试。
  - **签到**: 无，来不来都行。
  - **平时作业**: 作业每次五题。
  - **期中考试**：无原题，但是比较常规，只考点集拓扑，期中后开始学习代数拓扑。
- **期末闭卷考试 (50%)**: 正常考试，分为判断题，简答题和证明题。
  - **判断题**: 直接判断对错。
  - **简答题**: 可以缩略详细过程的证明题，例如计算基本群。
  - **证明题**: 共三题，均无原题。包含道路连通，同伦等内容。同调群内容并未涉及。
