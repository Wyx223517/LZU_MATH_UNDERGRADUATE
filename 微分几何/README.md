## 📝 成绩构成

本课程的最终成绩由以下三部分构成，具体占比没有算。

- **平时分**: 作业+大作业，大作业是思考题，应该是挑三四题就可以。
- **期中考试**: 计算和证明。证明会有一个书上定理和一个新题。
- **期末考试**：题型和期中一致。
